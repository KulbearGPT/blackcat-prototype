后台权限层级

权限采用向下继承：高级别自动拥有低级别的全部执行权限。L3 可以处理 L2 的日常操作，L4 可以处理 L1–L3 的全部操作；高额操作由 L4 直接执行，不要求另一名人员复核。
L1 基础客服执行联系、核对、内部备注、流程暂停和提交审批；无权执行资金及高风险操作。
L2 客服主管处理团队任务、常规取消、转派和限额内审批。
L3 运营负责人管理目录、用户与陪玩状态、推荐绑定和高金额操作。
L4 管理员 / 所有者继承全部低级权限，管理权限与 Discord 身份映射，并可直接执行高额操作；仍不能硬删除账务事实。
推荐关系隐私规则：系统禁止自荐。被推荐用户无法查看返佣对象、比例或金额；仅受益人本人及具备相应权限的管理人员可查看相关记录。
